TP noté de langage et compilation. 
Ce programme comporte plusieurs partie : 
-La lecture d'un fichier contenant une grammaire
-La création d'un automate a partir de cette grammaire
-La création des premiers et suivant
-La création d'un tableau d'analyse
-L'execution sur un mot
Nous n'avons pas eu le temps de réalisé l'execution du programme ni de réalisé les test sur les dernières étapes.

Afin que le programme fonctionne correctement, il faut qu'un fichier nommé "grammaire.txt" placé a l'emplacement de l'éxécution (avec le .exe si le programme est lancé depuis le .exe,
sinon dans le dossier des .cpp si l'éxécution est lancé avec la compilation
