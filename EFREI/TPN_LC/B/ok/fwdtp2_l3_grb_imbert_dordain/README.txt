Projet Langage et compilation Imbert-Dordain

/!\ Ce projet ne compile qu'en forçant le compilateur à utiliser C++ dans sa version 11 /!\
Ce choix permet d'utilsier les nouvelles forme de boucles courtes de c++ qui allègent la syntaxe déja lourde de notre programme.
En cas d'impossibilité de compilation, une trace d'execution est fournie dans le fichier Imbert_Dordain_trace.txt.

		*Calculer les premiers ainsi que les suivants

		*Créer la table d'analyse

Nous n'avons pas été en mesure de réaliser la pile d'éxuction et donc la vérifcation d'un mot donné.
